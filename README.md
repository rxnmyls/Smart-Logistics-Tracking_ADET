# Smart-Logistics-Tracking_ADET

This project simulates IoT-generated data for a Smart Logistics Tracking system. It demonstrates how sensors are used to monitor and manage the movement and condition of packages in real time.

📦 Overview
The system generates simulated data representing logistics operations, including package location, identification, and environmental conditions. This helps illustrate how IoT improves visibility and efficiency in supply chain management.

⚙️ Features
GPS tracking (latitude and longitude)
RFID-based package identification
Temperature monitoring for goods (e.g., cold storage)
Delivery status updates (In Transit, Delivered, Delayed)
📊 Dataset Description
The generated dataset includes the following fields:

timestamp – time of data recording
package_id – unique package identifier
latitude & longitude – simulated GPS location
temperature – condition of the package (°C)
rfid_tag – RFID identifier
status – delivery status
🛠️ Tools Used
Python
Pandas
NumPy
Jupyter Notebook
GitHub
🎯 Purpose
This project aims to demonstrate how IoT data can be simulated and used in real-world logistics applications. It highlights the importance of real-time tracking, monitoring, and data-driven decision-making.

🚀 Application
Smart logistics systems like this are used by companies to:

Track shipments in real time
Ensure product quality during transport
Improve supply chain transparency and efficiency
